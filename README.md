## 🔧 Development Setup

To run locally:

1. Clone the repo:
    ```bash
    git clone https://github.com/Wandera-Martin/portfolio-website.git
    cd portfolio-website
    ```

2. Serve with a local dev server (recommended):
    ```bash
    python3 -m http.server 8000
    ```

3. Open in your browser:
    ```
    http://localhost:8000
    ```
